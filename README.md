# Mindy POS

Punto de Venta usando Electron y Vue.

### Instalación

```
$ npm install jfdelarosa/mindy
```


### Iniciar la app
Para iniciar la app se deben de abrir dos terminales, una para empezar el servidor, y otra para iniciar electron.

Terminal 1:

```
$ npm start
```

Terminal 2:

```
$ electron .
```
